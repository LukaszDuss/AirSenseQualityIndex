# AirSense Quality AI

Prognoza jakosci powietrza (ASQI) na bazie danych GIOŚ, Open-Meteo i modelu LSTM.

**Uruchomienie i instalacja:** zobacz [URUCHAMIANIE.md](URUCHAMIANIE.md)

Szybki start (Windows):

```text
setup.bat
run.bat
```

Potem: http://localhost:8501 (klient), http://localhost:8501/admin (panel).
